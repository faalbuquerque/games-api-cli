<h1>Consumindo API de Games</h2>

<h4>Consumindo API de games em Ruby pelo terminal.</h4>

<h2 align="center"> Tecnologias utilizadas:</h2>
<p align="center">
  &#10003; Ruby
  &#10003; net http
  &#10003; json
</p>

Clone o projeto:
```
git clone git@github.com:faalbuquerque/games-api-cli.git
```

Entre na pasta:
```
cd games-api-cli
```

Rode o programa:
```
ruby games.rb
```
